# PDF Summary Assistant (v1.0)

A minimal engineering-ready demo:
- Extract text from a PDF
- Generate a simple local summary
- Write outputs to files

## Requirements
- Python 3.x

## Run

### Basic
```bash
python src/main.py -i sample.pdf

Verbose logs
python src/main.py -i sample.pdf -v

Specify output directory
python src/main.py -i sample.pdf -o output
